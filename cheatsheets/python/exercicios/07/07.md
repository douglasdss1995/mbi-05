# Exercício 7: Classificador de Notas

**Descrição**:

Crie uma função chamada **classificar_nota** que recebe uma nota (0-100) e retorna o conceito:

* A: 90-100
* B: 80-89
* C: 70-79
* D: 60-69
* F: 0-59
* "Inválido" para valores fora do intervalo

```python
classificar_nota(95)
# "A"

classificar_nota(82)
# "B"

classificar_nota(59)
# "F"

classificar_nota(105)
# "Inválido"

classificar_nota(-5)
# "Inválido"
```
